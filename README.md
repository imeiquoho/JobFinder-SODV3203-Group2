# 🚀 JobFinder

**JobFinder** is a clean and simple Android app that helps users explore job opportunities, save favorites, and keep notes while applying.

Built as part of our **Mobile Application Development** course, this app focuses on usability, modern UI, and real-world features.

## ✨ Features

- 🔍 Browse job listings  
- 🔎 Search by title, company, location, or skills  
- 🧰 Filter jobs (Full-Time, Remote, Contract, Part-Time)  
- ↕️ Sort by salary or title  
- ❤️ Save / Unsave jobs  
- 📁 View saved jobs in a separate tab  
- 🔄 Switch between **List** and **Grid** view  
- 📄 View detailed job descriptions  
- 📝 Add personal notes for each job  
- 👆 Track apply button clicks  
- 🎨 Modern UI using **Material 3**

## 📱 App Screens

- **Job List** → Browse, search, filter, and sort jobs  
- **Saved Jobs** → View saved jobs only  
- **Job Details** → Full job info (salary, description, etc.)  
- **Job Notes** → Add and manage personal notes
